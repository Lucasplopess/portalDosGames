Portal dos Games
=================

Bem-vindo ao Portal dos Games — um site simples em HTML/CSS/JS que apresenta diversos gêneros de jogos (RPG, FPS, etc.) de forma visual e responsiva.

Visão Geral
-----------
O objetivo deste projeto é criar um site com navegação intuitiva, usando boas práticas de HTML, CSS e JavaScript, onde os visitantes possam conhecer diferentes estilos de jogos e descobrir conteúdo relevante. A página inicial tem cards para cada gênero, e há páginas separadas para “RPG”, “FPS” e “Sobre”.

Tecnologias Utilizadas
----------------------
- HTML5  
- CSS3 (incluindo estilos responsivos)  
- JavaScript (para a navegação mobile/hamburger menu)  
- Estrutura de diretórios simples:
  
      /css  
      ├ style.css  
      └ styleglobal.css  
      /img  
      ├ portal-dos-games-logo.png  
      ├ the-witcher-3.png  
      └ codwarzone.png  
      /js  
      └ mobile-navbar.js  
      index.html  
      rpg.html  
      fps.html  
      sobre.html  
      contato.html

Estrutura do Projeto
--------------------
Arquivo/Pasta        | Função  
---------------------|---------------------------------------------  
`index.html`         | Página inicial com banner, seção de gêneros e cards de jogos.  
`rpg.html`           | Página dedicada ao gênero de RPG.  
`fps.html`           | Página dedicada ao gênero de FPS.  
`sobre.html`         | Página sobre o portal, explicando objetivo e autor.  
`contato.html`       | Formulário ou informações para contato.  
`/css/`              | Pasta de estilos CSS.  
`/img/`              | Pasta de imagens utilizadas no site.  
`/js/`               | Pasta de scripts JavaScript para funcionalidades extras.

Como Executar Localmente
------------------------
1. Clone este repositório:  
     `git clone https://github.com/Lucasplopess/portalDosGames.git`  
2. Vá para o diretório do projeto:  
     `cd portalDosGames`  
3. Abra o arquivo `index.html` no seu navegador de preferência.  
4. Caso deseje, você pode servir o projeto via um servidor local (ex: “Live Server” no VS Code) para testar funcionalidades de mobile/hamburger.
